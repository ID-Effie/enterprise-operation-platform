# Enterprise Operation Platform

一个基于 Vue 3、TypeScript、Vite 的中后台练习项目。

## 技术栈

- Vue 3
- TypeScript
- Vite
- pnpm

## 当前进度

- Day 1：完成项目初始化
- 已清理默认页面
- 已建立基础目录结构

## 目录结构

- src/views：页面组件
- src/components：通用组件
- src/router：路由配置
- src/stores：状态管理
- src/api：接口请求
- src/utils：工具函数
- src/types：类型定义
- src/styles：全局样式

## 启动方式

```bash
pnpm install
pnpm dev
```
